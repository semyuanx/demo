# 面试小demo
## 
## 项目描述
###### 采用 react、react-router、react-redux、redux、superagent(ajax)、superagent-jsonp(jsonp获取数据)
## 
## 项目运行
###### `git clone https://github.com/semyuanx/demo.git`
###### `cd demo`
###### `npm i`
###### `npm start`
## 
## 项目打包
###### `npm run build`
## 
## 项目总结
1. 接口逻辑和页面逻辑不统一，因此耗费时间比较长。
2. 因工作原因，没有时间对页面进行细化
3. 页面逻辑较少，但还是采用的redux。对以后的扩展应该有好处。
